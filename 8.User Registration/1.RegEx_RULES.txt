^ - Start of String
$ - End of String or Line
. - Any Character
+ - 1 or more of a character
\ - Special characters
? - 0 or one character
* - 0 or more of a character

{3} - A certain number range
{2,4} - range 2 to 4
{2,} - 2 to N
[ ] - Matches any of the character in a set


shopt -s extglob - To turn on
